AWS Latency Testing - 111221007

This project is a web-based tool designed to measure network latency to various AWS regions worldwide. It provides users with a simple interface to test the AWS latency from the local machine.

✨ Features

  • 🌐 Real-time latency testing to AWS regions
  • 📊 Interactive charts showing latency over time
  • 🏆 Top 3 fastest regions ranking
  • 🇹🇼 Taiwan region included for local testing

🛠️ Technology Stack

  • ⚡ Angular 18 with Server-Side Rendering
  • 🎨 Bootstrap 5 for responsive design
  • 📈 ngx-charts for data visualization
  • ☁️ Heroku for cloud deployment

How it works

The application sends HTTP requests to AWS S3 buckets in different regions and measures the response time. This gives you an accurate measurement of network latency from your location to various AWS data centers.
